##### 1.How to link git to local and commit changes

&nbsp;	1.create a local file aa.txt

&nbsp;	2.git init (iniatilize git)

&nbsp;	3.git add aa.txt (add txt file)

&nbsp;	4.git commit -m "any message" (commit your change)

&nbsp;	5.git remote add origin https://github.com/vivek-kodepaka/spring-core-fundamentals.git (connect to remote git and give credentials)

&nbsp;	6.git remote -v (verify git)

&nbsp;	7.git branch -M main (branch)

&nbsp;	8.git push -u origin main (push changes from local to main
